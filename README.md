# AI Ethics RAG Model

## Overview
This project is a Retrieval-Augmented Generation (RAG) system built using Flowise.

It answers questions based on uploaded documents related to:
- AI Ethics
- Bias in AI
- Responsible AI

## Tools Used
- Flowise
- Mistral AI
- Google Gemini Embeddings

## Workflow
1. Upload document (PDF)
2. Split text into chunks
3. Convert into embeddings
4. Store in vector database
5. Retrieve relevant data
6. Generate answer using LLM
